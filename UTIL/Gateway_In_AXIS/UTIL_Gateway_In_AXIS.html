Gateway In AXI Stream

Library

HDL/Interfaces

Description

This block serves as a utility block to simplify the connection between an HDL block with an input AXI Stream port, which comprises three signal lines: the tdata and tvalid signals going in one direction, and the tready signal going in the opposite direction. Note that in the generated HDL code, the block name will be used as the name for the AXIS port.
