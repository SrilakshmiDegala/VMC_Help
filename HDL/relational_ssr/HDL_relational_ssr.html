Vector Relational

Description

The Vector Relational block implements comparator for vector inputs.

Parameters

Parameters specific to the Vector Relational block are:

Comparison

Specifies the comparison operation computed by the block.

Super Sample Rate (SSR)

This configurable GUI parameter is primarily used to control processing of multiple data samples on every sample period. This block enables 1-D vector data support for the primary block operation.

Other parameters used by this block are explained in the topic Common Options in Block Parameter Dialog Boxes.
